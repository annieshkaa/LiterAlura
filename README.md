# LiterAlura 
Challenge de desarrollo de un catálogo de libros usando lo aprendido en la especialización de programación BackEnd de Alura. 

En este emocionante desafío de programación, desarrollé LiterAlura, un catálogo de libros construido en Java que permite interactuar con los usuarios mediante consola.

El proyecto integra consumo de API, manipulación de datos JSON y persistencia en base de datos, ofreciendo una experiencia completa de desarrollo backend.

🎯 Objetivo del Proyecto

Desarrollar un Catálogo de Libros que:

Permitir interacción textual (vía consola) con los usuarios.

Consumir datos desde una API de libros.

Procesar respuestas en formato JSON.

Almacenar información en una base de datos.

Permitir consultar y filtrar libros y autores.

🛠️Funcionalidades

✔️ Búsqueda de libros mediante API
✔️ Conversión y análisis de datos JSON
✔️ Almacenamiento en base de datos
✔️ Consulta de libros registrados
✔️ Consulta de autores
✔️ Filtros personalizados
✔️ Interacción completamente vía consola

🚀 Tecnologías Utilizadas

Java

API REST

JSON

Base de datos relacional

JPA 

PostgreSQL
